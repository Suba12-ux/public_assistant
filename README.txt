Данный проект создан для предоставления помощи при создании характеристик по шаблону.

СТРУКТУРА ФАЙЛОВ|___________________________________________________________
--ФАЙЛ upchar содержит структуру файлов в основном формата csv с готовыми шаблонными высказваниями их можно пополять собстванными или заменить на другие.
--ФАЙЛ data_profiles.csv содержит шифрованй формат данных вашего профиля менять ничего нежелательно, создается автоматически.
--ФАЙЛ GenderData содержит данные гендера его маожно менять если система дала ложный вывод на субъект.
--ФАЙЛ os_group текстовый файл уже с готовой табличной структурой.

СТРУКТУРА инициализации программы|___________________________________________________________
1.Запустить файл start_shell.bat
2.В поле ключ нужно ввести свой уникальный ключ или придумать если запуск происходит первый раз.
3.Нажать на кнопку найти профиль. Если профиль не найден будет инициализирована регистрация.
----3.1 После регистрации необходимо еще раз нажать найти профиль.
4.Ввести url адрес группы.
5.Дождаться завершения.
